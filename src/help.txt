prj: prj [-cph] [file]
  Manage your projects with the use of .prj files

  You can make these .prj files by yourself or with
  the help of this command line tool

  Options:
   -v  --verbose   Verbose mode
   -b  --brief     Brief mode
   -e  --err       Checks a project file for errors
   -p  --parse     Parses a project file and processes it
   -h  --help      Show this help menu
